# Display Library

### Libreria di controllo ed interfacciamento con il display (specificatamente *Nextion*)

Il display touchscreen consente il controllo diretto dell'ESP32 permettendo funzioni quali l'implementazione del controllo manuale e della modifica e lettura dei parametri interni ad ogni Asset ARM device

-------------------

## Library index
* ### [DisplayInterface](DisplayInterface.md)
* ### [NextionUtils](NextionUtils.md)

## Library Fratures

- ### L'interfaccia *[DisplayInterface](DisplayInterface.md)* consente la pre-definizione di controller strutturalmente organizzati nel display, in modo di rendere facilmente accessibile l'intruduzione di nuovi pulsanti e elementi grafici che necessitano della gestione eventi click
